# who-is-that-pokemon

A simple pokemon quiz made using vuejs.

#Screenhots
![Screenshot](1.png)


## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
